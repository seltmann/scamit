SCAMIT members have an occasional need to exchange files that are larger than can be accomodated by email. The commercial Dropbox service can provide an accessible way to do that.

Dropbox (acessible at http://www.dropbox.com) provides two main services: remote storage of files and synchronization of files across computers. Remotely stored files can be shared between users (in a controlled way). The file synchronization capability can be extremely convenient and useful for people who want to work on the same files or documents across multiple machines or locations. However, for this purpose (file exchange), the multiple-machine synchronization can be ignored.

The Dropbox system can be accessed either by: (1) installing software on your local computer, which allows for both file upload/download and synchronization across computers; or (2) through the web browser interface, which allows for file upload/download. You can selectively share files with specific other Dropbox users, or make files public and send recipients a web link to retrieve the file.

Here's a quick guide to getting started with Dropbox:

  1. Visit http://www.dropbox.com
  2. Click “Log in” at the top right of the page.
  3. Select “Create an account” on the next panel.
  4. Follow the instructions to enter your email address and a password (NOT the password to your email account, but a new password for Dropbox).

  6. If you can install software on your computer:
    1. Install and run the Dropbox application (accept the defaults along the way).
    2. The new folder “Dropbox” will be synchronized across all your machines.
    3. Create a new folder inside the “Dropbox” folder.
    4. Drag files into the folder.
  7. To share the folder with other Dropbox users:
    1. [From the web page] Hover over folder, select far-right triangle, and use “Invite to folder”.
      [Dropbox application] Right-click on the folder and share it.
    2. Recipients will receive an email telling them about the share.
  8. To share a file with anyone (not just Dropbox users):
    1. Put the file in the “Public” folder.
    2. [From the web page] Hover over filename, select far-right triangle, and select “Copy public link”.
      [Dropbox application] Right-click the filename and select “Copy public link”.
    3. Send recipients the link, from which they can download the file.
